## Top 2\. Kostenloser Open-Source-DVD-Ripper - Handbrake

Handbrake kann nie aus der Liste der besten DVD Ripper Freeware vermissen. Obwohl Handbrake nur [DVD in MP4](https://tools.techidaily.com/macxdvd/products/) und MKV rippen kann, erfüllt es fast allen allgemeinen Bedarf, da MP4 mit fast allen Geräten kompatibel ist. ### So rippen Sie DVDs kostenlos mit Handbrake?

**Schritt 1.** HandBrake downloaden und installieren. Zuerst downloaden Sie Handbrake 1.6.1\. Die neueste Version kann unter macOS 10.13 und höher, Windows 11/10 und höher, anderen Plattformen funktionieren. 

**Hinweise:** Laden [Sie zunächst Handbrake v1.6.1 hier kostenlos herunter](https://handbrake.fr/). Auf einem Windows-PC werden Sie möglicherweise mit "_To run this applicatoni, you must install .NET._ " aufgefordert, folgen Sie den Anweisungen und laden Sie die .NET-Datei herunter, um die neueste Handbrake auszuführen. Handbrake 1.6.1 erfordert Microsoft .NET Desktop Runtime 6.

**Schritt 2.** Laden Sie libdvdcss für Handbrake herunter, um geschützte DVDs zu rippen. Handbrake kann nur DVDs rippen, die KEINE Form von Kopierschutz enthalten. Wenn Ihre Discs mit CSS oder Regionalcode verschlüsselt sind, installieren Sie besser Handbrake libdvdcss auf Mac/PC. 

![DVD Ripper Freeware Handbrake](https://www.macxdvd.com/tutorial-de/../mac-dvd-video-converter-how-to/article-image/homebrew-libdvdcss.jpg) 

**Notiz:** Die libdvdcss-Bibliothek konzentriert sich darauf, Handrabke beim Rippen von DVDs mit CSS-Schutz zu unterstützen. Wenn es um fortschrittlichere Verschlüsselungstechnologien wie Sony ARccOS oder Disney X-Project DRM geht, kann Handbrake diese Verschlüsselung immer noch nicht umgehen. Sie benötigen [**MacX DVD Ripper Pro**](https://tools.techidaily.com/macxdvd/products/), um DVD-Verschlüsselungstechnologien auf höherer Ebene zu handhaben.

**Schritt 3.** Laden Sie die Quell-DVD in Handbrake. Legen Sie die Quell-DVD in das optische Laufwerk Ihres Mac/PC ein und starten Sie HandBrake. Sobald die Quelldaten geladen sind, sehen Sie das Titel-Dropdown-Menü in der oberen linken Ecke. Handbrake wählt standardmäßig den Haupttitel aus.

**Schritt 4.** Wählen Sie das Ausgabeformat. Handbrake bietet Ihnen 3 Ausgabeformate MP4, MKV und WebM. Die Standardeinstellung ist MP4 H264\. Sie können Ihren gewünschten mit bevorzugtem Codec (x264, x265, vp8, vp9) auswählen.

![handbrake reißt kopiergeschützte dvd](https://www.macxdvd.com/tutorial-de/article-image/hb-mac-format.jpg) 

Darüber hinaus bietet es eine Fülle von voreingestellten Profilen, aus denen Sie auswählen können, Apple-Geräte, Android-Geräte, gängige 1080p/720p 30FPS-Profile, Roku, PS4 usw. 

**Schritt 5.** DVDs jetzt kostenlos rippen. Tippen Sie oben links in der Version von Handbrake für Mac auf die Schaltfläche "Codieren starten", um DVDs auf dem Mac zu rippen. Wenn Sie eine Quelle mit mehreren Titeln haben, können Sie diese mit der Schaltfläche "Zur Warteschlange hinzufügen" in die Warteschlange stellen. Sobald Ihre Warteschlange fertig ist, klicken Sie auf die Schaltfläche "Start", um den Batch-DVD-Ripping-Prozess auf dem Mac mit Handbrake zu starten.
