**1\. Create New Project** 

In case you're familiar with Photoshop, you know what layer means. In iMovie, Project is similar to layer in Photoshop. Thus, prior to import MP4 into iMovie, create a new project first. Click "Create New" icon to create one project.

**2\. Import MP4 Files into iMovie**

To add MP4 into iMovie, here are three ways to go. 

* Tap "Import" button at the top of the interface to import MP4 to iMovie.
* Drag and drop MP4 files to the area as the right picture shows.
* Browse inside your own personal library via "My Media" to import MP4 files into iMovie.
